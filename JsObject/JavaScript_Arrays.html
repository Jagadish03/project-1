JavaScript Arrays

Arrays are used to store multiple values of same type as well as different type
In array values will be stored in index postions.
The values of array is also known as elements of the Array.
The first element will be stored in the 0th index postion and the last element will be stored in lenth-1.
The JS array is a collection of both homogeneous and hetrogeneous elements.
